## 中文

本次发布修复了 Word 文档中引用格式修复可能反复写入字体格式的问题。

### 修复内容

- 保持插件启动后不自动扫描或修复当前文档。
- 颜色、字号和下划线修复现在会先检查当前格式，只有格式确实不一致时才写入 Word 文档。
- 减少撤销栈中反复出现 `VBA-Font.Color`、`VBA-Font.Underline` 等记录的情况。
- 降低打开大型 Zotero 文档后由后台格式写入造成的卡顿风险。

### 说明

完整的链接引用和修复格式仍然需要用户主动点击插件按钮才会执行。

## English

This release fixes repeated font-format writes during citation formatting repair in Word documents.

### Fixes

- Keeps the add-in from automatically scanning or repairing the active document after startup.
- Color, font size, and underline repairs now check the current formatting first and only write to Word when a change is actually needed.
- Reduces repeated `VBA-Font.Color` and `VBA-Font.Underline` entries in Word's undo history.
- Lowers the risk of sluggish behavior caused by background formatting writes in large Zotero documents.

### Note

Full citation linking and formatting repair still run only when the user explicitly clicks an add-in button.
